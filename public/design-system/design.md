# Shift Design System

Companion document for the in-app **Design System** (`/design-system`) in the Shift product suite.

## Contents in the app

- **Foundations** — Color, typography, spacing, layout, and elevation tokens.
- **Components** — Buttons, inputs, tables, dialogs, navigation, and other UI primitives.
- **Patterns** — Composed layouts and flows (page chrome, tables, uploads, empty states, and more).

## Repository references

- Design system audit and section inventory: `docs/design-system-audit.md`
- Architecture and repo layout: `docs/architecture/repo-structure.md`
- Design tokens (Figma exports): `design-tokens/` (consumed by generation scripts into CSS primitives)

## Brand assets

Sample logos and marks used in product flows live under `public/media/brand-samples/` and `public/media/logos/`. The **Download brand guidelines** package on the Design System home page bundles representative brand imagery for offline use.
